# 🤖 Agente Personalizado para WhatsApp  

Um bot para WhatsApp criado com **[Baileys](https://github.com/WhiskeySockets/Baileys)** que atua como **agente personalizado**, permitindo responder mensagens, executar comandos e integrar com serviços externos.  

## ✨ Funcionalidades  
- 📩 Responde automaticamente mensagens recebidas  
- 🔘 Suporte a botões interativos e mensagens com menus  
- ⚙️ Fácil de personalizar para diferentes finalidades (FAQ, suporte, assistente pessoal, etc.)  
- 🛡️ Estrutura preparada para autenticação e middlewares  

## 🛠️ Tecnologias  
- [Node.js](https://nodejs.org/)  
- [Baileys](https://github.com/WhiskeySockets/Baileys)  
- JavaScript
